**Client:** React, Redux, Material UI, more... 




## Run Locally

Clone the project

```bash
  git clone https://github.com/eyosiyas-js/EMAIL-UI.git
```

Go to the project directory

```bash
  cd EMAIL-UI
```

Install dependencies

```bash
  yarn 
```

Start the server

```bash
  yarn start
```
